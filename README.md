# Experiment-8-Temperature

Aim

To create an Android application that converts temperature from Celsius to Fahrenheit and Fahrenheit to Celsius.

Algorithm

Start Android Studio.

Create a new Android project.

Select Empty Activity template.

Open activity_main.xml file.

Add EditText fields to enter temperature value.

Add two RadioButtons (C→F and F→C) for selection.

Add a Button to perform conversion.

Open MainActivity.java file.

Write logic to convert temperature based on selected option.

Run the application and display the converted temperature.
